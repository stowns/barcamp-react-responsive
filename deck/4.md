# What Do I Want

```jsx
class MyComponent {
  
  renderUsernameInput() {
    return <input className="username"></input>
  }
  
  renderMobile() {
    return (
      {this.renderUsernameInput()}
    )
  
  }
  
  renderDesktop() {
    return (
      <p>User Name: {this.renderUsernameInput()}</p>
    )
  }
  
}
```

```css
.username {
  color: red;
}
```
